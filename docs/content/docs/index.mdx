# Lamport One-Time Signatures

Quantum-resistant digital signatures using only hash functions. This library provides production-ready Lamport OTS for Solidity (EVM) and Go.

## Features

- **Quantum Resistant**: Security based on hash preimage resistance (Keccak256)
- **EVM Compatible**: Works on any EVM chain without precompiles
- **Gas Optimized**: Assembly-optimized verification (~100k gas)
- **Threshold Support**: T-of-N signing via MPC coordination

## Quick Start

```bash
# Install with Foundry
forge install luxfi/lamport

# Run tests
cd contracts && forge test -vvv
```
